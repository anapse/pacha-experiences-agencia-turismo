## ✦ Sobre el Proyecto

**Pacha Experiences** es una plataforma web profesional llave en mano para la gestión completa de agencias de turismo, diseñada específicamente para el mercado peruano. Ideal para agencias en Huacachina, Cusco, Arequipa y destinos turísticos en todo el Perú.

---

## ✨ Características Principales

### 🌐 Sitio Web Público
- 🎠 Carrusel hero con loop infinito y auto-play
- 🔍 Catálogo de experiencias con búsqueda, filtros y ordenamiento
- 📅 Calendario de reservas con disponibilidad en tiempo real
- 📋 Páginas de detalle con galería, itinerario y reseñas
- 💬 Chat flotante multicanal en tiempo real
- 🌍 Multi-idioma (Español / Inglés)
- 💰 Multi-moneda (Soles, Dólares, Euros)
- 🌙 Modo oscuro / claro
- 📱 Diseño responsive mobile-first

### ⚙️ Panel Administrador
- 📊 Dashboard con estadísticas en vivo
- 🏜️ Gestor completo de experiencias (CRUD)
- 📋 Sistema de reservas y cupos
- 👥 CRM de clientes con historial
- 🚗 Gestión de conductores y operadores
- 🚐 Administración de flota de vehículos
- 💰 Facturación y reportes
- 🎠 Editor de carrusel y contenido visual
- 🔧 Configuración general y redes sociales
- 📈 Reportes con gráficos y exportación

### 🔧 Tecnología
- Frontend moderno con React 19 + Vite
- Backend robusto con Node.js + Express
- Base de datos SQLite (migrable a PostgreSQL)
- Chat en tiempo real con Socket.IO
- API REST documentada
- Despliegue con Cloudflare Tunnel
